# 企业级销售采购一体化智能体平台

本目录用于交付一套可上线、企业级、覆盖销售与采购核心流程的智能体项目方案，包含：

## 第一阶段：核心交付

- `02-产品文档PRD.md`
- `03-技术方案文档.md`
- `04-实施方案.md`

## 第二阶段：高级交付
- `05-高保真原型与页面设计说明.md`
- `06-数据字典与指标口径文档.md`
- `07-系统集成接口设计文档.md`

项目定位：
构建“销售+采购”双域协同的企业级智能体平台，统一承载以下四类业务智能体能力：

1. 销售线索筛选与跟进智能体
2. 招投标/方案生成智能体
3. 供应商评估智能体
4. 采购询价与比价智能体

目标是打造一套具备：
- 可上线落地
- 可配置、可审计、可扩展
- 安全合规
- 支撑企业级集成与运营
- 达到业内最顶尖水准

的完整解决方案。

建议阅读顺序：
1. 先看 `01-项目总方案.md`
2. 再看 `02-产品文档PRD.md`
3. 然后查看 `03-技术方案文档.md` 与 `04-实施方案.md`
4. 若进入投标、汇报、研发、测试、上线阶段，再继续查看 `05` 到 `10` 系列文档
